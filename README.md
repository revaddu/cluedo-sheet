# cluedo-sheet

The Cluedo Sheet - An interactive detective sheet for Cluedo / Clue board game

# Development

## Project Setup

```sh
npm install
```

## Compile and Hot-Reload for Development

```sh
npm run dev
```

## Type-Check, Compile and Minify for Production

```sh
npm run build
```

## Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
